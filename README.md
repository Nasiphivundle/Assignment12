# Assignment12

Project Description

The Smart Student Task Manager is a web-based system created to help students manage their academic work more easily. The system allows students to create tasks, set deadlines, receive reminders, and keep track of completed work.

This project was developed throughout different software engineering assignments. During the project, different concepts such as system analysis, Agile planning, UML diagrams, REST API development, testing, and GitHub project management were applied to improve the system step by step.

Project Objectives

The main goals of this project are to:

Help students organize their academic tasks and assignments
Reduce missed deadlines
Improve time management skills
Allow students to track completed and pending tasks
Apply Agile software development methods in a practical project


System Features

The system includes the following features:

User registration and login
Creating tasks
Editing tasks
Deleting tasks
Setting deadlines
Viewing all tasks
Marking tasks as completed
Sending reminders before deadlines
